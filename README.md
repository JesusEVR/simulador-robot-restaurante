# Simulador en terminal de un robot mesero y cocinero

Este programa en Java permite gestionar pedidos de hamburguesas en el restaurante ficticio ***McBurguesas***, esto a través de un robot que desempeña los roles de mesero y cocinero, mediante un sistema de menús y estados.

## Estructura del proyecto
- `menus/`: contiene las clases relacionadas con los menús a los que tiene acceso el robot
- `hamburguesas/`: define los distintos tipos de hamburguesas
- `estados/`: gestiona los estados del robot
- *Principal.java*: punto de entrada del programa

## Cómo correr el programa

Nos situamos en la carpeta `src` y ejecutamos los siguientes comandos para compilar los archivos y comenzar la simulación:

```
javac Principal.java
```


```
java Principal
```
